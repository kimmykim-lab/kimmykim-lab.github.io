Netflix Recommendation System

Summary: Explored how Netflix recommends content by analyzing its hybrid algorithm, combining content-based and collaborative filtering. Focused on user behavior tracking and the structure of movie metadata.

Tools & Methods: Metadata analysis, Diagram modeling, Behavioral data interpretation

Key Insight: Netflix does not rely on explicit user input; instead, it tracks user behavior (watch history, duration, clicks) to dynamically adjust genre classification and deliver personalized recommendations.

Online Retail SQL Database Project

Summary: Developed a fully normalized (3NF) relational database for a women’s clothing & accessories e-commerce platform. The project included ERD design, SQL table creation, data import, view generation, and Tableau-based sales analysis.

Tools & Methods: SQL Server, DBeaver, ERD (Crow’s Foot Notation), Tableau, Excel

Key Insights: Sales peaked in Q2 2024 with strong demand for seasonal apparel. High return rates due to “Wrong Size” issues correlated with low average ratings (2.33), indicating user experience challenges.

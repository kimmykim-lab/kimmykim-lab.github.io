Summary: Proposed a joint language exchange initiative between San José State University (SJSU) and San José Public Library (SJPL) to host weekly conversation circles in Vietnamese, Spanish, and Mandarin. The program would foster cultural exchange, improve language fluency, and strengthen campus-community collaboration.
